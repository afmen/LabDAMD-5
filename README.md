# LabDAMD-5
Lista de Compras Simples em Flutter


Comandos Essenciais
# Criar projeto
flutter create nome_projeto

# Executar app
cd lista_compras_simples
flutter run

# Executar no navegador
cd lista_compras_simples
flutter run -d chrome

# Parar execução
Ctrl + C (no terminal)

# Reload quente (durante execução)
r (no terminal) ou Ctrl+S (no VS Code)
